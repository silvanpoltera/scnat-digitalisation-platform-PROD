★ Hauptempfehlung · GPU-Server
01 Whisper Large v3 Turbo
Gratis · MIT-Lizenz · Shared Server für alle
Mit einem dedizierten GPU-Server läuft Whisper als lokale REST-API, die alle MacBook-User der SCNAT über das Netz nutzen können – ohne GPU auf dem eigenen Gerät. faster-whisper reduziert VRAM-Bedarf auf ~3 GB. Bei CH-DE als Kernbedarf: das FHNW-Fine-Tune direkt auf demselben Server deployen.
Sofort · Überbrückung bis Server
02 töggl.ch
CHF 29/Mt. · Daten in CH · kein Setup
Bis der GPU-Server läuft: töggl.ch als schlüsselfertige Lösung. Daten bleiben in der Schweiz, CH-DE funktioniert nativ, kein IT-Aufwand. Kann parallel weiterlaufen für Nutzer ohne Zugriff auf den internen Server.
Auf MacBook · Ohne Server
03 Whisper via MLX / whisper.cpp
Gratis · Apple Silicon · Metal-Beschleunigung
Auf M1/M2/M3/M4 MacBooks läuft Whisper über MLX (Apple) oder whisper.cpp mit Metal-Beschleunigung direkt auf dem Gerät – ohne Cloud, ohne GPU-Server. Langsamer als dedizierter Server, aber für gelegentliche Nutzung durchaus praktikabel.

whisper.cpp – läuft nativ auf Apple Silicon via Metal. Kein Python, kein Setup-Aufwand. Direkte CLI oder als API. Modell (~1.6 GB) einmalig herunterladen.
MLX Whisper (Apple Framework) – Python, läuft auf dem Neural Engine des M-Chips. Schneller als whisper.cpp auf M3+. Installierbar via pip install mlx-whisper.
Faustregel: M2 Pro reicht für gelegentliche Nutzung (Interviews, kurze Sitzungen). Für Dauerbetrieb oder mehrere Nutzer gleichzeitig: dedizierter Server.
Schweizerdeutsch – realistische Einschätzung
Kein kommerzielles Modell unterstützt CH-DE wirklich nativ. Whisper transkribiert Schweizerdeutsch, obwohl es nicht explizit Teil der Trainingsdaten war — es behandelt es wie eine Übersetzung ins Hochdeutsche. Das klappt für viele Zwecke. Wer präziser sein will, nimmt das FHNW/Flurin17-Fine-Tune (SwissDial ETH, Swiss Parliaments Corpus V2 FHNW 293h, ArchiMob UZH) – läuft auf demselben Server oder MacBook. töggl.ch bleibt die einzige schlüsselfertige SaaS-Option mit echter CH-DE-Unterstützung und Schweizer Datenhaltung.
